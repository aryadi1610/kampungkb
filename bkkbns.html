Variabel Input

Keberadaan Pokja (kepemilikan pokja, SK Pokja, Pokja terlatih, sosialisasi pokja)



Sumber dana (APBN, APBD, Dana Desa, CSR, swadaya masy, Hibah, Lainnya)



Kepemilikan poktan (BKB, BKR, BKL, UPPKS, dan PIKR)



Ada sarana tempat sekteraiatan dan sarana rumah data



Keberadaan PKB/PLKB dan Regulasi

Variabel Proses

Penggunaan data untuk perencaan antara lain Data PK, Data Rutin, Data podes, Data Sektoral, Sumber Data lainnya



Mekanisme operasional antara lain : rapat perencaan kegiatan koordinasi dengan lintas sektoral melakukan sosialisasi kegiatan, melakukan monev, membuat laporan.



Intervansi lintas sektoral : Keterlibatan sektor lain dalam pelaksaan kegiatan



Variabel Output

Partisipasi keluarga / remaja dalam kegiatan Poktan BKB, BKR, BKL, PIK-R dan UPPKS



Capaian CPR kontrasepsi modern



Capaian MKJP kontrasepsi Mix



Persentase PUS tidak berKB / unmet need
